# Data Sources

## 1. data.gov.sg (Phase 1)
- **Data**: Private residential transactions, price indices, CEA records
- **Access**: Free, no API key needed
- **Format**: REST API, JSON responses
- **Usage**: Primary transaction data, price index tracking

## 2. OneMap API (Phase 1)
- **Data**: Geocoding, nearby amenities (MRT, schools, malls, parks)
- **Access**: Free
- **Usage**: Address geocoding, walking distance calculations for nearby amenities

## 3. URA Data Service API (Phase 2+)
- **Data**: Transactions, rental data, vacancy rates
- **Access**: Requires access key from developer.tech.gov.sg
- **Usage**: Supplementary transaction data, rental comparisons

## 4. Web Sources (Phase 3)
- **Sources**: Developer sites, NewLauncher.sg, PropertyGuru, EdgeProp
- **Method**: AI agent web search + extraction
- **Usage**: Project details, pricing updates, launch information
